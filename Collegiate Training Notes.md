Pages / Templates
- Home - quite fixed, mostly uses set ACF fields in set positions, you fill it out like a form. DOESN'T use content section
- General / Default - completely flexible, uses the content section mainly, but has some fields for the header
- Landing Page - at this stage an organisational template and intended to be used with the 'cards' block
	- We do suggest keeping the header design consistent with your 'depth' on the landing page (ie, 1st level maroon, 2nd cream)
- Events - no content, just the events coming up 

Gutenburg Blocks
- go to kitchen sink to go through them all

Post Types:
- Students - used on the student carousel on the home page. That opens a modal with some copy and a video
- Events
	- These are edited much like pages, with all the blocks we discussed from above
	- You do add a date (they are only shown if they are in the future) and a category (for filtering on the events page)
- Team Members - very similar to students, except rendered via the 'Team Card' block
- History - very similar to team members, but utilised by the 'History' block
